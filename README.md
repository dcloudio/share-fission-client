## 一、前端功能模块
### 1. 基础功能
- 登录、注册、系统设置等，直接基于 uni-starter 实现

### 2. 分销推广体系 - 推广工具
用户在“分销中心”页面可操作的功能：
1.  专属推广二维码：生成带用户ID参数的海报图片，支持保存到相册
2.  专属邀请码：查看唯一的数字/字符邀请码
3.  推广链接 (URL)：支持复制链接

### 3. 分销推广体系 - 我的团队
- 团队列表：查看自己邀请的用户（头像、昵称、注册时间）
    - 层级：一级用户（直接邀请）、二级用户（间接邀请）
- 收益展示
    - 时间维度：今日、昨日、近 7 天、全部
    - 项目维度：积分、新增下线数（区分：1 级 2 级）

### 4. 积分奖励 - 积分提现
- 发起提现申请
- 查看积分状态（未提现、已提现）
- 查看提现规则（手续费比例、积分兑现金比例）
- 提现方式（银行卡、支付宝（开转账API））

### 5. 积分商城系统
#### 5.1 积分获取相关
- 看广告获取积分（前端展示积分到账）
- 每日签到获取积分（可多次看广告次数，但每日有次数限制）

#### 5.2 商品展示与兑换
1.  积分商品列表：按积分高低排序、分类筛选（仅虚拟商品）
2.  商品详情页：
    - 展示轮播图、详情描述
    - 展示兑换所需积分
3.  兑换流程：
    - 确认兑换并扣除积分
    - 查看兑换订单状态（可查看卡密和兑换地址）

## 二、后端（管理端）功能模块
面向管理员/系统逻辑的功能，包含数据处理、规则配置、管理操作

### 1. 分销推广体系 - 关系绑定逻辑
- 绑定时机：新用户注册/首次登录时绑定上下级
- 绑定关系管理：管理员后台可解除

### 2. 积分奖励规则设置
- 配置看广告积分分配比例：
  单次广告产生积分（在非实时模式下是固定的值，可设置，在实时模式下，根据实时返回的金额*100）
    - 当前用户积分比例
    - 一级直推用户积分比例
    - 二级直推用户积分比例
- 配置提现规则：
    - 提现手续费比例（如20%）

### 3. 积分商城系统 - 商品管理 CRUD
1.  新增：上传商品图片、填写名称/库存/所需积分/兑换限制
2.  查询：商品列表，按状态（上架/下架）搜索
3.  编辑：修改库存、积分数值、详情图
4.  删除/下架：软删除或下架，前端隐藏
5.  卡密管理：批量导入商品卡密

### 4. 订单模块
- 查看积分兑换记录
- 查看订单对应的卡密信息

### 5. 数据统计
- 时间维度：今日、昨日、近7日、总计
- 项目维度：广告收益、积分增加、积分消耗、用户新增、用户登录
